# Welcome {-}
This is a draft version of the first chapters of the upcoming book on geospatial data visualization with the R-package tmap.
We stil have to write chapters about interactive maps, small multiples, animations, shiny integration, and guidelines to make good maps.

The content of the current version of this book is based on **tmap** version 3.x, which is stable. Currently we are working on a major update of tmap (version 4), which will probably be released early 2022. The book will be updated accordingly. The biggest change of tmap v4 will be the extendability: it will be easy to create new map layer types, for instance bivariate choropleths and donut maps (https://dashboards.cbs.nl/v1/commutingNL/).

This version of the book was built on 2021-08-27.
